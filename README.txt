	|	Instrucciones:													|
	|																	|
	|	1. Descargar archivo de Aula y descomprimir.					|
	|																	|
	|	2. Abrir 3 terminales en la carpeta donde se encuentran			|
	|		los archivos archivos descomprimidos.						|
	|																	|
	|	3. Ejecutar en terminal 1 el siguiente comando:					|
	|																	|
	|		>> go run servidor-gato.go									|
	|																	|
	|	4. Esperar a que envíe mensaje de inicio.						|
	|																	|
	|	5. Ejecutar en terminal 2 el siguiente comando:					|
	|																	|
	|		>> python3 servidor-intermediario.py						|
	|																	|
	|	6. Esperar a que envíe mensaje de Inicio.						|
	|																	|
	|	7. Ejecutar en terminal 3 el siguiente comando:					|
	|																	|
	|		>> python3 servidor-cliente.py								|
	|																	|
	|	8. Jugar en el servidor cliente.								|
	|																	|
	+-------------------------------------------------------------------+
	|																	|
	|	Nota: antes de ejecutar los archivos, se debe tener instalado	|
	|		  Python3 y Golang.											|
	|																	|
	|	Nota 2: Esta tarea fue ejecutada en Ubuntu usando las versiones	|
	|			3.7.8 de Python y 1.13.8 de Go.							|

	|	Nota 3: Se da por hecho que el usuario va a escoger casillas	|
